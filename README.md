# The Reset — Personal Habit & Life Tracker

Dark-mode habit tracker with sober counter, daily check-ins, streaks, charts, books, and weekly summaries. 100% localStorage, no backend.

**Stack:** Next.js 16 · TypeScript · Tailwind v4 · Recharts · Framer Motion · date-fns · canvas-confetti

```bash
pnpm install
pnpm dev
```

Deploy: `pnpm build && pnpm dlx vercel`
