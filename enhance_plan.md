## Phase 2 — AI / OpenAI Features

### Daily Debrief
- Button on the Today page: "Debrief today"
- Sends today's full `DayLog` (habits, sleep, energy, mood, note) + deep work time + task completion to GPT-4o
- Returns a short personal narrative analysis
- Example output: *"You hit 9/12 habits. Energy was 6/10 — 1.5 below your 7-day average. Deep work fell 20 min short of goal. Missing sleep cutoff appears on your last 4 low-energy days — worth watching."*
- Route: `POST /api/ai/debrief`

### Weekly Review Summary
- Button on the Weekly page: "Generate review"
- Sends 7-day `DayLog` array + deep work totals + book pages read
- GPT-4o narrates the week: trends, best day, worst day, habit consistency, sleep patterns
- Output saved as a Note automatically

### Chat With Your Data (`/ask` page)
- Chat UI with message thread
- User asks anything: *"How was my sleep last week?"* / *"What day did I do the most deep work?"* / *"Am I reading more this month?"*
- API route fetches relevant slice of Supabase data, builds a context string, calls GPT-4o, streams response
- Streaming via `ReadableStream` so replies feel instant
- Most powerful feature — one assistant that knows all your personal stats

### AI Note Summarizer
- On Notes and Files pages: "Summarize" button
- Sends note/file content to GPT-4o
- Returns: key points (bulleted), action items, one-line summary
- Useful for project plans and reports stored in Files

### Task Suggestion
- On Today page: "Suggest tasks" button
- Sends last 3 days of incomplete tasks + today's date + any notes from this week
- GPT-4o returns 3 prioritized task suggestions
- User can one-click accept them into today's task slots

### Pattern Insight (weekly, auto)
- Runs silently once a week (triggered by visiting Charts)
- Looks for correlations in your data: workout → better mood next day, poor sleep → low deep work, etc.
- Shows a small "Insight of the week" card on the dashboard

## Phase 3 — Gamification

### XP + Level System
- Every action earns XP:
  - Habit completed: +10 XP each
  - Full habit day (all 12): +50 bonus XP
  - Deep work session logged: +1 XP per minute
  - Task completed: +15 XP (critical), +10 (important), +5 (nice)
  - Book finished: +200 XP
  - Note written: +20 XP
  - Streak milestone hit: +100–500 XP
- Level thresholds: 0, 500, 1500, 3000, 5000, 8000, 12000...
- Level shown in sidebar next to name
- Level-up animation (confetti already in project!)

### Achievement Badges
A collection of unlockable badges stored in Supabase (`achievements` table):

| Badge | Condition |
|---|---|
| First Light | Complete Fajr 7 days in a row |
| Iron Mind | 14 consecutive full habit days |
| Deep Work God | 4h+ deep work in a single day |
| Page Turner | Read 100 pages in a week |
| Library Card | Complete 5 books |
| Night Owl Slayer | In bed by target 10 nights in a row |
| Quran Keeper | Quran session 30 days straight |
| Unstoppable | 30-day streak on any habit |
| Centurion | 100-day sober streak |
| Flow State | 3h deep work every day for a week |
| Daughter Day | Daughter time logged 14 days straight |
| Clean Week | All 7 days with 10+ habits in a week |
| Book Worm | 500 total pages read |
| Deep Thinker | 50 total deep work hours |

- Badge gallery page showing locked/unlocked status
- Toast notification on unlock
- Badges shown on a profile/stats card

### Daily Score Upgrade
- Current score is already calculated — turn it into a visible grade: S / A / B / C / D
- S tier requires: 10+ habits + 2h deep work + all tasks done + sleep logged
- Grade shown large on Today page with colour coding

### Weekly Leaderboard vs Yourself
- Compare this week's score to last week's score
- "You're up 23 points from last week" / "Down 14 — bounce back"
- No external competition needed — just you vs your best self

### Streak Multipliers
- If you're on a 7+ day streak, habit completions give 1.5x XP
- 14+ day streak: 2x XP
- Makes maintaining streaks feel rewarding mechanically

### Daily Challenge
- Each day shows 3 random bonus challenges, e.g.:
  - "Complete all 12 habits today" (+100 XP)
  - "Log 3h of deep work" (+80 XP)
  - "Write a note today" (+30 XP)
- Resets at midnight
- Stored in `daily_challenges` table, generated by a simple seeded random function

### Boss Days
- Every 7th day of a streak is a "Boss Day" — all habits must be completed to keep the streak alive
- Special visual on the Today page when a Boss Day is active
- Adds tension and makes weekly rhythm feel meaningful

### Profile / Stats Card
- `/profile` page showing:
  - Current level + XP bar
  - Total deep work hours (all time)
  - Total books read
  - Total notes written
  - Longest streak ever (per habit)
  - All earned badges
  - "Member since" date
  - Personal bests: best habit day, best deep work day, best reading week

## Phase 4 — App Enhancements

### Goal Setting (`/goals`)
- Monthly goals with progress tracking:
  - Deep work hours target (e.g. 40h in April)
  - Books to read (e.g. 2 books)
  - Habit rate target (e.g. 85% habit completion)
- Progress bar on dashboard
- Stored in `goals` Supabase table

### Carry-Over Tasks
- Tasks not completed by end of day get flagged with `carriedOver: true`
- On the next day's task slot, show "Carried from yesterday" in amber
- The `carriedOver` field already exists in the Task type — just needs UI

### Pomodoro Mode on Focus Page
- Toggle between Stopwatch and Pomodoro (25/5 cycle)
- Count completed pomodoros for the day
- Pomodoro count stored alongside deep work sessions
- Ring/sound on cycle end (optional)

### Individual Habit Streaks
- Current streaks are sober-counter only
- Show each habit's current streak and best streak in the Streaks page
- Highlight habits with broken streaks in a "Recovery" section

### Journaling Prompts on Today
- Below the note field, show a random daily reflection prompt:
  - "What was the hardest moment today?"
  - "What would make tomorrow better?"
  - "What are you grateful for?"
- Prompts rotate daily from a fixed list

### Reading Goals
- Per-book page target (e.g. "50 pages/day")
- On the Books page: show "On track" / "Behind" / "Ahead" badge on each active book
- Weekly reading summary in Weekly page

### PWA (Progressive Web App)
- `manifest.json` + service worker via `next-pwa`
- "Install to home screen" on mobile
- Works offline for viewing cached data
- Makes it feel like a native app on your phone
- Push notifications for daily habit reminders

### Mobile Bottom Navigation
- On small screens, replace the sidebar with a bottom tab bar
- 5 tabs: Today, Focus, Streaks, Charts, More
- Much better ergonomics for phone use

### Dark / Light Mode Toggle
- Theme already uses CSS variables — easy to add
- Toggle in sidebar, persisted to Supabase user preferences

### Keyboard Shortcuts
- `T` → go to Today
- `F` → go to Focus
- `N` → new note
- `Ctrl+K` → command palette (search across notes + files + pages)

## Phase 5 — Notifications & Automations

### Daily Email Digest (Resend API)
- Every morning at 7am: email with yesterday's score, today's habit list, a motivational message
- Every Sunday: weekly summary email with charts as embedded images

### Browser Push Notifications
- "Don't forget to log today's habits" at 9pm if not yet logged
- "Deep work reminder" at 10am
- Uses the Web Push API + a simple cron via Vercel Cron Jobs

### Vercel Cron — Weekly XP Tally
- Runs every Sunday midnight
- Calculates weekly XP total, checks for new badges, sends email digest
- `vercel.json` cron configuration + a `POST /api/cron/weekly` route
